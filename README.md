# Python DSA Documentation

This repository contains documentation and examples for Python Data Structures and Algorithms and ML Engineer interview preparation guide.

## Documentation

The documentation is built using MkDocs and is available at: https://kunaltembhare003.github.io/python_DSA/

### Local Development

To work on the documentation locally:

1. Install MkDocs and the Material theme:
```bash
pip install mkdocs-material
```

2. Serve the documentation locally:
```bash
mkdocs serve
```

3. Build the documentation:
```bash
mkdocs build
```

The documentation will be automatically deployed to GitHub Pages when changes are pushed to the main branch.
